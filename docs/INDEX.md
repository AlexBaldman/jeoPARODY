## 🤝 Contributing to Documentation

When adding new documentation:

1. **Place it in the correct category** (Planning, Session, Feature, Technical)
2. **Update this INDEX.md** with a link
3. **Cross-reference related docs** using relative links
4. **Follow naming conventions:**
   - `UPPERCASE_WITH_UNDERSCORES.md` for major docs
   - `Feature-Name-Spec.md` for feature specs
   - `SESSION_N_SUMMARY.md` for session summaries
