
# 🗺️ JeoPARODY Roadmap

This document outlines our vision for JeoPARODY's growth beyond the MVP.

---

## 🚀 MVP (Milestone 1)
- [x] Single-player random trivia
- [x] AI host dynamic rewording of questions
- [x] Firebase Auth + High Score tracking

---

## 🌱 Phase 2 – AI Study Buddy
- [ ] Pause gameplay to chat with AI
- [ ] Generate and save study guides
- [ ] Search topics within game

---

## 👥 Phase 3 – Multiplayer
- [ ] 1v1 Player Challenges
- [ ] Leaderboards & Achievements
- [ ] Real-time group trivia (Study Groups)

---

## 📚 Phase 4 – User-Generated Content (UGC)
- [ ] Import Anki decks, PDFs, and text files
- [ ] AI-assisted trivia deck creation
- [ ] Community Deck Library & Sharing

---

## 🛍️ Phase 5 – Monetization
- [ ] Premium subscription features
- [ ] Cosmetic store (host skins, themes)
- [ ] Trivia Deck Marketplace
- [ ] Education licensing for classrooms

---

## 🧠 Future Features
- AI voice synthesis for host personalities
- VR/AR trivia experiences
- API for 3rd-party trivia apps
- Twitch/YouTube Live integration for streamers

---

## 📆 Target Timeline
| Quarter       | Milestone                             |
|---------------|---------------------------------------|
| Q3 2025       | MVP Closed Beta                       |
| Q4 2025       | Public Launch + Study Buddy           |
| Q1 2026       | Multiplayer + UGC Tools               |
| Q2 2026       | Marketplace + EdTech Expansion        |
